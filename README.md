# l33t speak converter

A simple utility to convert text into l33t speak.

## Install
```
$ npm install leet-speak-converter
```
Or if you prefer using Yarn:
```
$ yarn add leet-speak-converter
```

## Usage
to run CLI version (in command line) use:
```
$ npm run start:cli
```
Or if you prefer using Yarn:
```
$ yarn run start:cli
```
